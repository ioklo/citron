# 전역 변수 선언하기

[[root-namespace|최상위 네임스페이스]]에서 지역 스코프 없이 변수 선언을 하면 [[global-variable|전역 변수]] 선언이 됩니다. 

아래 예제에서 a는 전역 변수이고, b는 지역 변수입니다.
```
int a = 0;

{
    int b = 0;
}
```

전역 변수 공간 할당은 [[선언 스캐닝 시점]]에 이뤄지고, 일단 0에 해당하는 값으로 메모리를 채웁니다. [[실행 시점]] 초반에 전역변수 초기화가 일어납니다.
```
int a = func();
int b = a + 1;
```
