# 전역 변수 참조

현재 컨텍스트에서 이름이 겹치지 않는다면, 거의 모든 경우에서 이름으로 전역 변수를 참조할 수 있습니다. 현재 컨텍스트에서 id의 이름을 찾는 방법은 [[query-id|id 참조]]를 참고하세요.

[[full-qualified-id]]을 통해서 접근할 수 있습니다.

```csharp
int a = 7;

class C
{
    void F()
    {
        print(a);        // 전역 변수

        int a = 12;
        print(::a); // 글로벌 변수 참조용
    }
}
```
